# 🤖 NEURAL CHAT - 革新的なAIチャットボット

## 概要
最新のAI技術を搭載した次世代チャットボットインターフェース。直感的な操作と美しいニューラルネットワーク背景アニメーションで、AIとの対話を新たな体験に変革します。

## 🚀 主要機能

### 🧠 AIチャット機能
- **リアルタイム会話**: 自然な対話形式でのAI応答
- **パーソナリティ選択**: 4つの異なるAIキャラクター
- **コンテキスト理解**: 会話履歴を踏まえた応答
- **多言語対応**: 日本語・英語・韓国語・中国語

- **テーマ切替**: ダーク・ライトモード対応
- **レスポンシブデザイン**: デスクトップ・モバイル完全対応

### 🎙️ 高度な入力機能
- **音声認識**: Web Speech API活用
- **音声フィードバック**: リアルタイム音声可視化
- **ファイル添付**: 画像・文書ファイル対応
- **自動補完**: インテリジェント入力サポート

### ⚙️ カスタマイズ機能
- **回答設定**: 長さ・速度・スタイル調整

## 🎮 操作方法

### 基本操作
- **メッセージ送信**: Enter キーまたは送信ボタン
- **改行**: Shift + Enter
- **音声入力**: マイクボタンをクリック
- **設定変更**: 歯車アイコンから設定画面

### ショートカット
- **新しいチャット**: Ctrl + N
- **設定を開く**: Ctrl + ,
- **履歴クリア**: Ctrl + Delete
- **テーマ切替**: Ctrl + T
